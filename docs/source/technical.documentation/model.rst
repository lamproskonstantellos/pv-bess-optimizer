Model overview
==============

See :doc:`mip_formulation` for the constraint set,
:doc:`regulatory_framework` for the self_consumption / merchant mode definitions,
and :doc:`objectives` for the profit-maximisation objective.

The authoritative mathematical specification (every constraint,
objective, settlement and KPI as a numbered equation mapped to its
implementing symbol) is the domain design document set under
``docs/`` (``self_consumption_design.md``, ``merchant_design.md``,
``balancing_market_design.md``, ``ppa_design.md``,
``economics_design.md``, ``uncertainty_design.md``), indexed with a
shared notation table by ``docs/README.md``.
