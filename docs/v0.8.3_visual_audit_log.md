# v0.8.3 visual audit log

The addendum prescribes a manual visual audit of every PDF produced
by `python main.py inputs/input.xlsx --solver highs` against the
universal rules.  This file is the per-run record.

## How to fill it in

1. Run the optimiser with the default workbook
   (`python main.py inputs/input.xlsx --solver highs`).
2. Open every PDF under
   `results/<run>/04_financial_plots/`,
   `results/<run>/05_energy_plots/`,
   `results/<run>/06_uncertainty_plots/`.
3. For each PDF, tick the row below.  If any check fails, capture a
   screenshot, add a row to the "Failures" section, and fix before
   declaring round-3 complete.

## Checklist per PDF

* [ ] No text overlaps a data line or bar
* [ ] No text touches an axis spine
* [ ] No italic free-floating prose captions
* [ ] No white-edged markers
* [ ] Monthly SOC plot has DD-MM-YYYY axis + right-side `SOC (%)`
* [ ] Yearly SOC plot has MM-YYYY axis + right-side `SOC (%)`
* [ ] LCOE/LCOS summary reports every value through its legend
      (no in-axes annotations, no diamond markers)
