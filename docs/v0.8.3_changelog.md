# v0.8.3 — plotting universality round

Round-3 sweep across every public plotting function so the rules
that were ad-hoc style preferences in v0.8.1 / v0.8.2 are now
mechanically enforced for every existing and future plot.

The contract is enumerated in
[`tests/test_plotting_universality.py`](../tests/test_plotting_universality.py)
and [`tests/test_grep_audits.py`](../tests/test_grep_audits.py); they
together cover all 34 public plot functions and refuse to pass when
a forbidden pattern reappears.

## 1. Universal helpers

`pvbess_opt.plotting.style` gains two helpers:

* `annotate_value_safe(ax, x, y, text, …)` — single entry point for
  every bbox-wrapped numeric annotation.  Accepts a `transform` for
  axes-fraction anchors (top-right summaries) or data coordinates
  with an optional `offset_points=(dx, dy)`.
* `expand_axes_for_annotations(ax, pad=0.05)` — widens xlim / ylim
  so annotations placed near the data edges do not clip the frame.

Every plotting module routes its bbox annotations through these
helpers; no module retains an inline `ax.text(... bbox=...)` or
`ax.annotate(... bbox=...)` call.

## 5. Round-3 audit log (30 plotting functions)

Every function in the addendum's inventory now passes the eight-item
sweep checklist.  Functions touched directly:

* `plot_revenue_stack_yearly` — italic caption + white marker edge
  removed.
* `plot_lifetime_cycles` — bar colour centralised, "Total: N cycles"
  routed through `annotate_value_safe`.
* `plot_lcoe_lcos_summary` (+ `_draw_benchmark_row`) — full
  redesign per Task 2 / 3 / 4.
* `plot_npv_waterfall` — NPV total annotation via helper.
* `plot_npv_tornado`, `plot_irr_tornado` — dumbbell endpoint labels
  via helper.
* `plot_daily_soc`, `plot_monthly_soc`, `plot_yearly_soc` — SOC
  colour centralised; monthly / yearly gain `twinx()` right
  %-axis; monthly gains `_setup_day_axis` x-axis.
* `plot_lifetime_summary` — 4 inline hex literals replaced.
* `plot_rolling_horizon_distribution`,
  `plot_foresight_gap_comparison` — histogram colour centralised.
* `plot_input_forecast_band`, `plot_input_seasonal_boxplot` — panel
  colours centralised.

Functions that already satisfied every rule:
the daily / monthly / yearly supply / surplus / combined trios, the
merchant dispatch / revenue trios, `plot_cumulative_cashflow`,
`plot_yearly_cashflow_bars`, `plot_payback`,
`plot_monthly_cashflow_year1`, and `plot_dam_intraday_heatmap` — all
verified by the new universality suite.
